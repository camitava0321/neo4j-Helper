Neo4J-Cypher Query Language (CQL)

A query language for Neo4j Graph Database.

Neo4j CQL Commands/Clauses

The following are frequently used Neo4j CQL Commands/Clauses

  • CREATE
  • MATCH
  • RETURN
  • WHERE
  • DELETE
  • REMOVE
  • ORDER
  • SET

CREATE To create nodes, relationships and properties

CREATE Nodes

create (:Organisation {acronym:"SWR",name:"Swiss Reinsurance", type:"ReInsurance"})
create (n:News {text:"Market Share of National Insurance has increased", sentiment:0.8})

CREATE Constraint

create constraint on (o:Organisation) assert length(o.acronym)=3
create constraint on (o:Organisation) assert o.acronym is unique
create constraint on (o:Organisation) assert exists(o.type)

CREATE Relationship (no attributes) between existing nodes

match (n:Organisation {acronym:"SWR"}) match(i:Organisation) where i.acronym="NIA" create (i)-[:INSURED_WITH]->(n)
match (n:Place) match(i:Organisation) where i.name<>"The Telegraph" create (i)-[:ORG_IN]->(n)

CREATE Relationship (with attributes) between existing nodes

match (n:Organisation {acronym:"SWR"}) match(i:Organisation) where i.acronym="NIA" create (i)-[:INSURED_WITH{expiry:"31/12/2016"}]->(n)

CREATE Relationship (with attributes) between new nodes

create (:Organisation {acronym:"LIC",name:"Life Insurance Corporation of India", type:"Insurance"})-[:INSURED_WITH{expiry:"31/12/2016"}]->(:Organisation {acronym:"MHR",name:"Munich Re Group", type:"ReInsurance"})

Should always be used with a RETURN/CREATE

Basic Match - match all

match (n:News) return n
match (n:News) return n.id,n.text

MATCH Relationship

match (a)-[:INSURED_WITH]-(b) return a,b

CREATE Relationship [Match to nodes then create a relation
